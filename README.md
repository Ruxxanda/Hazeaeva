## Локальный запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/ruxxanda/Hazeaeva.git
```

2. Перейдите в папку проекта:

```bash
cd Hazeaeva
```

3. Откройте файл `index.html` в браузере.
